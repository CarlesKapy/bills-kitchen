Development Package for creating VMs with Vagrant and Chef

All you need for cooking with Chef and Vagrant (on Windows).

Contents

 |-- boxes          => the Vagrant base boxes that we reference in the .Vagrantfile
 |-- docs           => various documentation artifacts, shortcuts to online documentation of the provided tools
 |-- home           => %HOME% folder where .gitconfig, VirtualBox VMs, .vagrant.d and other user-specific stuff goes to
 |-- install        => files that need to be installed manually such as the MONACO.TTF font
 |-- repo           => the chef-repo and separate repositories for the cookbooks
 `-- tools          => preinstalled tools, such as:
        |-- console2        => a better command prompt for windows (with tabs, better copy & paste, etc..)
        |-- notepad++       => text editor (our IDE)
        |-- p4merge         => diff/merge tool to be used when using `git difftool ...` or `git mergetool ...`
        |-- portablegit     => git installation from portable apps
        |-- putty           => in case you want to ssh / scp using putty
        |-- sshwindows      => ssh.exe ported to windows, required to make `vagrant ssh` work on windows
        `-- vagrant         => a devkit enhanced Ruby 1.9.3 with the following gems pre-installed:
                                * vagrant (patched to make `vagrant ssh` work on windows)
                                * chef
                                * sahara (patched to work on windows)
                                * chefspec
                                * foodcritic
                                * librarian

Installation

You only need to install VirtualBox from here, everything else comes already preinstalled.

Usage

Mount the DevPack to the W:\ drive by double-clicking the mount-w-drive.bat file. Then open a command prompt (either cmd or W:\tools\console2\Console.exe) and call the W:\set-env.bat script. This will set up your environment so that you have all the required tools in the PATH and you are ready to GETTING_STARTED with Chef, Vagrant and Co.
